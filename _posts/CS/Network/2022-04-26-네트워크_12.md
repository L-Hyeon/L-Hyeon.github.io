### Network Id, Host Id

- 클래스 별로 netId, hostId부분이 다름
- 클래스 A는 바이트 1만 netId
- 클래스 B는 바이트 2까지 netId
- 클래스 C는 바이트 3까지 netId

- NetId를 부여해서 Host를 구분
- NetId를 얻으면 hostId부분을 분배
- Class A의 NetID 하나에 붙어있는 HostId는 1600만개 정도 == 너무 많아서 대부분 낭비되고 있음
