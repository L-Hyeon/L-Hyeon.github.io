# 두 수의 곱 (~˘▾˘)~

### 문제

정수 num1과 num2가 주어질 때, num1과 num2를 곱한 값을 return하도록 soltuion 함수를 완성해주세요.

### 제한사항

0 ≤ num1 ≤ 100
0 ≤ num2 ≤ 100

### 입출력 예

|num1|num2|result|
|3|4|12|
|27|19|513|

### 풀이
```c++
#include <string>
#include <vector>

using namespace std;

int solution(int num1, int num2) {
    return num1 * num2;
}
```

```python
def solution(num1, num2):
    return num1 * num2
```
