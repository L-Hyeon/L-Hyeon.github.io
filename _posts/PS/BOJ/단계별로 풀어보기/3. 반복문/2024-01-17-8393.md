# 합

### 문제

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

### 입력

첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

### 출력

1부터 n까지 합을 출력한다.

### 예제

##### 예제 입력 1

```
3
```

##### 예제 출력 1

```
6
```

### 풀이

1부터 반복문을 돌면서 더해주면 되는 간단한 문제이다.

```python
import sys
input = sys.stdin.readline

def answer():
    s = 0
    for i in range(1, int(input()) + 1):
        s += i
    print(s)

answer()

```
